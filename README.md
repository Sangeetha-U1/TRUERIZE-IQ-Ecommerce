# 🛍️ Truerize IQ – Modern E-Commerce Fashion Website

Truerize IQ is a responsive and interactive front-end eCommerce website built using HTML, CSS, Bootstrap, and JavaScript.  
It includes product browsing, category filtering, shopping cart system, dark mode, and a simple login system using localStorage.

---

## 🏠 Pages Included

### 🏠 Home Page
- Hero banner with fashion branding
- Featured products section
- Navigation to all categories
- Smooth responsive layout

---

### 📦 Products Page
- Men, Women, Kids collections
- Product cards with image, price, quantity selector
- Add to cart functionality
- Category filtering system

---

### 👤 About Page
- Company introduction (Truerize IQ)
- Mission and vision
- Brand story and values
- Fashion and quality focus

---

### 📞 Contact Page
- Contact form (Name, Email, Message)
- Business contact details
- Email support information
- Phone support section

---

## ⭐ Testimonials Section
- Customer feedback display
- Product experience reviews

- Trust-building section for users

## 🚀 Features

### 🛒 Cart System
- Side sliding cart panel
- Add products with quantity
- Increase / decrease items
- Remove items
- Live total calculation
- Persistent storage using localStorage

---

### 📦 Product System
- Men / Women / Kids categories
- Product cards with images and pricing
- Quantity selector before adding to cart
- Smooth UI interaction

---

### 🔍 Filtering System
- Filter by category:
  - All
  - Men
  - Women
  - Kids
- Instant UI switching

---

### 🌙 Dark Mode
- Light / Dark theme toggle
- Saved in localStorage
- Applies across all pages:
  - Navbar
  - Cards
  - Cart
  - Background

---

### 🔐 Login System
- Simple frontend login (demo purpose)
- localStorage-based authentication
- Redirect protection for pages

---

### 📱 Responsive Design
- Fully mobile optimized
- Bootstrap grid system
- Responsive navbar
- Adaptive cart sidebar
- Works on all screen sizes
---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 4/5
- JavaScript (Vanilla JS)
- jQuery (for carousel)
- Font Awesome (icons)
- localStorage (data persistence)
